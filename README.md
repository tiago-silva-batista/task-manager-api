API RESTful de gerenciamento de tarefas desenvolvida com ASP.NET Core, Entity Framework Core e SQLite.

O projeto permite cadastro de usuários, autenticação com JWT e gerenciamento completo de tarefas.

---

## 🚀 Tecnologias Utilizadas

- C#
- ASP.NET Core
- Entity Framework Core
- SQLite
- JWT Authentication
- Swagger

---

## 🔥 Funcionalidades

- Cadastro de usuários
- Login com autenticação JWT
- CRUD completo de tarefas
- Proteção de rotas
- Integração com banco de dados SQLite
- Documentação automática com Swagger

## ▶️ Como Executar

### Clone o projeto

```bash
git clone https://github.com/tiago-silva-batista/task-manager-api.git
```

### Acesse a pasta

```bash
cd task-manager-api
```

### Restaure os pacotes

```bash
dotnet restore
```

### Execute o projeto

```bash
dotnet run
```
